# 003 — 北极星（唯一最终目标）

## 一句话

做出个人可用的**浏览器行动 Agent**：效果以**美团 Tabbit 1.0 公开披露的 Agent 任务成功率**为标尺，在用户**日常 Chrome 登录态**里用自然语言完成多步网页操作；**关键处一次批准**；**只有页面证据才算完成**；**中等模型（默认 MiniMax-M3）**跑正式分。壳与 Task 契约自建，**Nano 执行核不是终局**。

## 外部标尺：美团 Tabbit（证据级：公开报道 / 官方口径转述）

产品：**美团 · 光年之外（GN06）· Tabbit AI 浏览器**（2026-03 公测，2026-06 前后 1.0 上线）。

公开数字（媒体与官方介绍转述，**非我方复测**）：

| 指标 | 披露数值 | 含义（对方口径） |
|---|---|---|
| Agent 任务成功率 | 公测初 **53.1%** → 1.0 约 **91.8%** | 智能体任务整体成功（含跨场景工作流，不仅点网页） |
| 自研 Benchmark · 网页操作 | V0.23 **61%** → V1.0 **70%** | 网页操作任务成功率 |
| 自研 Benchmark · 总评分 | 65.0 → **94.3** | 对方综合分，细节未完全公开 |

参考来源（公开网页，非学术论文）：美团新闻稿/经济参考网转述（91.8%）、36Kr 等对 1.0 的报道（网页操作 70%、总分 94.3）。

**我们的对齐方式（严格）：**

1. **主对齐数：Agent 任务成功率 ≥ 91.8%**  
   在我方**已冻结的黄金旅程集合**上，用 **verified_pass / 有效 attempt** 计算，正式分必须用 **MiniMax-M3（或同级中等模型）**。  
   我方「成功」定义**严于**「看起来做完了」：必须满足可观察证据 + 0 假完成 + 外部提交 0 未批准。

2. **底线对齐数：网页操作成功率 ≥ 70%**  
   若扩展到黄金旅程以外的长尾站，网页操作 verified 成功率不得低于对方披露的 **70%** 网页操作线。

3. **不抄对方黑盒分母。** 对方 91.8% 的任务分布未完整公开；我们用**固定协议 + 可复现脚本/手工表**，数字可审计。打平的是**准确率水平**，不是照抄任务包。

### 操作换算（n=10 时）

| 旅程 | 对齐 91.8% 的操作标准 |
|---|---|
| Fixture 表单 / 媒体 | **连续 ≥10/10**（100%，避免小样本虚高） |
| 飞书 / B 站固定协议 | **≥10/10** 或累计 **n≥50 且成功率 ≥91.8%**；阶段性探路可报 9/10，**不得**宣称已对齐 Tabbit |
| 长尾网页操作池 | **≥70%** verified（对齐对方网页操作线） |

原先 PRD 的「真实站 8/10（80%）」**抬高**：对外宣称对齐美团时，**以 91.8% 为准**，80% 只算内部里程碑，不算北极星完成。

## 完成时用户可见的样子

1. 侧栏只见「你 / 助手」、目标、状态、下一步；不见 Planner/Navigator/step_failed。
2. 飞书类表单：填完 → 提交前停住等批准 → 批准后只提交一次 → 成功证据 → 完成回执。
3. B 站类媒体：播放后「暂停这个」绑同一对象 → 真暂停 → 新回执。
4. 成功任务可存本地 Skill，换输入后重跑。
5. 未批准绝不外部提交；证据不足不得显示已完成。

## 完成定义（闸门 G1–G7，含 Tabbit 对齐）

全部满足才算本阶段北极星完成：

| # | 闸门 | 标准 |
|---|---|---|
| G1 | 本地 fixture 表单 | **连续 10/10** verified；批准前 0 提交；批准后 1 提交；0 假完成 |
| G2 | 本地 fixture 媒体 | **连续 10/10**；暂停绑原媒体；0 假完成 |
| G3 | 飞书黄金旅程 | **≥91.8%** verified（n≥50 或 10/10）；0 未批提交；0 假完成 |
| G4 | B 站黄金旅程 | 同上；暂停绑定在有效 attempt 内达标 |
| G5 | 中等模型 | G1–G4 正式分母 = MiniMax-M3（或同级）；禁止旗舰独占刷分 |
| G6 | 核与壳 | 可替换执行核上线；Nano Planner 环可拔；L4 契约权威 |
| G7 | 隐私 | 非用户聊天存储无表单值/凭证/正文；无 replay 原始动作键 |
| **G8** | **Tabbit 对齐声明** | 公开发布/对内「达标」材料中写明：对比的是 Tabbit **91.8% Agent 成功率** 与 **70% 网页操作**；附我方分母、n、模型、失败分类；**不得**暗示已复现对方内部 Benchmark |

## 明确不是目标

- 做成 Tabbit 整浏览器或抄妙招广场。
- 只报「感觉差不多」而无 91.8%/70% 数字。
- 用旗舰模型刷高成功率再对标中等模型产品。
- 在 Nano Core 上无限堆功能。
